Using Galton's height Galton, compute the answers to these questions about outliers using the 1.5 IQR rule of thumb and the outlier function.

  1. Which of these statements will compute the number of cases that are outliers with respect to height? (Assume that the data frame is named galton.)

    outlier(galton$height)
    table(outlier(galton$height))
    outlier(table(galton$height))
    subset(galton, outlier(galton$height))
    outlier(subset(galton, galton$height))

  2. How many of the cases are outliers in height ?

ANSWER:
table( outlier( galton$height ) )
FALSE TRUE
897 1

  1. How many of the cases are outliers in mother?

ANSWER: > table( outlier( galton$mother ) )
FALSE TRUE

  1. How many of the cases are outliers in father?

ANSWER: table( outlier( galton$father ) )

  1. Looking just at the cases where mother is an outlier, how many of the children involved (variable sex) are female?
